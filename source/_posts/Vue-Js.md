Vue 安装与新建项目
<!--more-->
## Vue 安装
```sh
# 安裝 Vue CLI 到全局
npm i vue-cli -g

# 创建一个基于 webpack 模板的新项目
vue init webpack <Project>

# 进入项目,安装依赖
npm install

# 运行
npm run dev
```
